Синхронизация дневника с Google Таблицей

Один раз настроив её, вы получите: автоматическую выгрузку всех записей в таблицу и возможность закидывать задачи из своего Excel прямо в приложение.

Шаг 1. Создайте таблицу

  1. Откройте sheets.new (лучше с компьютера — так удобнее).
  2. Назовите таблицу, например, «Мой дневник».

Шаг 2. Вставьте скрипт

  1. В таблице: меню Расширения → Apps Script.
  2. Удалите всё в открывшемся редакторе и вставьте код ниже (кнопка «Скопировать»).
  3. Нажмите значок дискеты (Сохранить).

Шаг 3. Опубликуйте скрипт

  1. Справа вверху нажмите Развернуть → Новое развёртывание.
  2. Значок шестерёнки → выберите тип Веб-приложение.
  3. «Выполнять от имени» — От моего имени.
  4. «У кого есть доступ» — Все. (Это обязательно, иначе приложение не сможет обращаться к таблице.)
  5. Нажмите Начать развёртывание, разрешите доступ своему аккаунту (Google предупредит, что приложение не проверено — нажмите «Дополнительные настройки» → «Перейти»). Это ваш собственный скрипт, он имеет доступ только к этой таблице.
  6. Скопируйте URL веб-приложения (заканчивается на /exec).

Шаг 4. Подключите в приложении

  1. В дневнике: вкладка Профиль → Подключить таблицу.
  2. Вставьте скопированную ссылку → «Подключить».
  3. Приложение сразу выполнит первую синхронизацию — в таблице появится лист «Записи» со всеми вашими записями.

Как перенести задачи из вашего Excel

  1. Откройте в таблице лист «Входящие».
  2. Вставьте задачи одним из двух способов:
    — просто список: тексты задач в столбец A, по одной на строку;
    — рабочая таблица целиком, вместе с шапкой «Задачи | Ответственный | Статус | Срок | Примечание» — скрипт сам её распознает: строки со статусом «выполнено» пропустит, а срок, ответственного и примечание допишет в текст задачи.
  3. В приложении: Профиль → Синхронизировать сейчас.
  4. Задачи появятся в телефоне (фильтр «Задачи»), а из «Входящих» переедут в лист «Импортировано».

Важно: лист «Записи» приложение переписывает целиком при каждой синхронизации — не редактируйте его вручную, правки затрутся. Всё новое добавляйте через лист «Входящие» или через само приложение. Ссылка Web App — секретная: любой, у кого она есть, сможет читать и менять эту таблицу. Никому её не передавайте.
